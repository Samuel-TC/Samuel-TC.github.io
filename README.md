## 📋 Descripción

Portafolio web profesional desarrollado con Angular 14. Diseño moderno y responsivo que presenta mis proyectos, habilidades, educación y formas de contacto.

## ✨ Características

- 🎨 Diseño moderno y responsivo con Bootstrap 5
- 🚀 Aplicación de una sola página (SPA) con Angular
- 📱 Totalmente adaptable a dispositivos móviles
- 🎯 Navegación fluida entre secciones
- 💅 Estilos personalizados con Sass
- ⚡ Carga rápida y optimizada

## 🛠️ Tecnologías Utilizadas

- **Frontend Framework:** Angular 14.0
- **Lenguaje:** TypeScript 4.7
- **Estilos:** Sass + Bootstrap 5.2
- **UI Components:** ng-bootstrap
- **Herramientas:** Angular CLI 14.0.1

## 📦 Instalación

### Prerrequisitos

- Node.js (versión 14 o superior)
- npm (versión 6 o superior)

### Pasos

1. Clona el repositorio:
```bash
git clone https://github.com/Samuel-TC/Samuel-TC.github.io.git
cd Samuel-TC.github.io
```

2. Instala las dependencias:
```bash
npm install
```

## 🚀 Uso

### Servidor de Desarrollo

Ejecuta el siguiente comando para iniciar el servidor de desarrollo:

```bash
npm start
```

La aplicación se abrirá automáticamente en tu navegador en `http://localhost:4200/`

Los cambios que realices se reflejarán automáticamente sin necesidad de recargar.

### Construcción para Producción

Para compilar el proyecto para producción:

```bash
npm run build
```

Los archivos optimizados se generarán en el directorio `dist/`

## 🧪 Testing

Ejecuta las pruebas unitarias con:

```bash
npm test
```

│   │   └── nav-bar/       # Barra de navegación
